# Dashboard CO₂ y crecimiento económico

## Ejecución en Colab

Sube `Dashboard_CO2_Profesional.ipynb` y ejecuta todas las celdas en orden.

## Ejecución local

```bash
pip install -r requirements.txt
python app.py
```

## Render

- Build command: `pip install -r requirements.txt`
- Start command: `gunicorn app:server`
